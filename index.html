GREAT!

(Gait Recognition Engine for Amazing Timesaving!)

What It's About

The GREAT Software, carefully crafted by our professional team of programmers, takes input from webcams/CCTV cameras/almost any working camera and feeds it to our AI system which analyses the movement and determines whose gait it is. For now, a face recognition software is in place instead, as this is still a work in progress and more time is needed to train our AI models.

Please allow camera access!

Note: System will currently only return "THET PAING MIN", "LUCAS WONG", "WONG KYLER", and "No person detected".
